# Compiler-RPN_Formatter
A Reversed Polish Notation Formatter, rewrite infix expression to suffix expressions.

# Input Format
Input is recognized as below. <br>
`# a = (a + b) / [c + {d * e}] - (t - a) #` <br>
The expression above will be rewritten as: <br>
`aab+cde*+/ta--=`
